Review Notes:
- Runs only on Shopee domains (tw, sg, my, ph, th, vn, id, br) as listed in host_permissions.
- tabs: used to detect the active Shopee tab, navigate Shopee search pages, and open result tabs only after user action.
- activeTab: used to start analysis from the tab the user is currently viewing.
- scripting: used to read Shopee page content (search results DOM) to organize same-store results.
- storage: used to save user settings and temporary results locally.
- All processing is local; no data is sent to any server.
